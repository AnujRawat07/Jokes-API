POST/jokes

Description

Creates a new joke.

Request Body

{ "joke": "Why did the developer go broke? Because he used up all his cache." }

Response (201 Created)

{ "id": 6, "joke": "Why did the developer go broke? Because he used up all his cache." }
PUT/jokes/:id

Description

Fully updates a joke using its ID.

Request

PUT /jokes/1 { "joke": "This joke has been completely updated." }

Response (200 OK)

{ "id": 1, "joke": "This joke has been completely updated." }
PATCH/jokes/:id

Description

Partially updates a joke.

Request

PATCH /jokes/2 { "joke": "This joke was partially updated." }

Response (200 OK)

{ "id": 2, "joke": "This joke was partially updated." }
DELETE/jokes/:id

Description

Deletes a joke permanently.

Request

DELETE /jokes/3

Response (200 OK)

{ "message": "Joke deleted successfully" }
